# My Zootopia V2

My Zootopia V2 is a Python project that fetches and displays information about animals using the API-Ninjas Animals API. It allows users to search for animals and get details like taxonomy, characteristics, and habitat.

## Installation

To install this project, clone the repository and install the dependencies:

```bash
git clone https://github.com/AlbertSauer/My-Zootopia2.git
cd My-ZootopiaV2
pip install -r requirements.txt
